# Pezesha Account Management API

A Spring Boot application that provides REST APIs for managing bank accounts and money transfers.

## Features

- Account Management (Create, Read, Update)
- Money Transfers between accounts
- Balance Inquiries
- Swagger API Documentation
- Error Handling
- Input Validation

## Technologies

- Java 17
- Spring Boot 3.x
- Spring Data JPA
- PostgreSQL
- Swagger/OpenAPI 3.0
- Maven
- Lombok
- JUnit 5
- Mockito

## Prerequisites

- JDK 17 or later
- Maven 3.6+
- PostgreSQL 12+
- Your favorite IDE (IntelliJ IDEA recommended)

## Setup

1. Clone the repository:
```bash
git clone https://github.com/arthuradinder/pezesha.git
cd pezesha
```
2. Configure the database in application properties
3. Build the project:
```mvn clean install```
4. Run the application
```mvn spring-boot:run```
